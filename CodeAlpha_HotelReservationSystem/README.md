# Hotel Reservation System

## Description

Hotel Reservation System is a Java console-based application developed to simplify hotel room booking and reservation management. The system allows users to view available rooms, book rooms, cancel reservations, view booking details, and simulate payments. It uses Object-Oriented Programming (OOP) concepts and ArrayList to manage hotel room information efficiently.

## Features

- View available hotel rooms
- Room categorization (Standard, Deluxe, Suite)
- Book a room
- Cancel room booking
- View booked rooms
- Payment simulation
- Customer information management
- User-friendly console interface
- Menu-driven application
- Prevent duplicate room bookings

## How It Works

1. Start the application.
2. Select an option from the main menu.
3. View available rooms.
4. Enter the room number to book.
5. Enter the customer name.
6. Complete the payment simulation.
7. View booking details.
8. Cancel a booking if required.
9. Exit the application.
